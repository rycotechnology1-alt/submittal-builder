# Phase 2 Bruno Collection

This collection exercises the Phase 2 CRUD flow against a running local web app:

1. Set `baseUrl` to `http://localhost:3000`.
2. Sign in through the app or auth API and copy the session cookie into a Bruno environment variable named `sessionCookie`.
3. Run requests in order:
   - `01 Create Project`
   - `02 Create Package`
   - `03 List Project Packages`
   - `04 Create Manual Item`

The collection assumes Phase 1 auth is already working and focuses only on the Phase 2 endpoints.
